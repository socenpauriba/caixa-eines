Caixa d'Eines és un plugin que ofereix una col·lecció de shortcodes útils per a WordPress.

== Descripció ==
Caixa d'Eines afegeix funcionalitats útils al teu lloc WordPress a través de shortcodes fàcils d'utilitzar. Actualment, inclou shortcodes per mostrar l'any actual i per calcular el temps de lectura estimat d'una publicació.

== Instal·lació ==
1. Puja la carpeta `caixa-eines` al directori `/wp-content/plugins/`.
2. Activa el plugin des del menú 'Plugins' de WordPress.

== Com Utilitzar ==
1. [any]: Aquest shortcode mostra l'any actual. Simplement afegeix `[any]` a qualsevol publicació o pàgina.
2. [temps_lectura]: Aquest shortcode calcula el temps de lectura estimat d'una publicació basat en el nombre de paraules. Afegeix `[temps_lectura]` a les teves publicacions per mostrar el temps de lectura estimat.
